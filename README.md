# djstudio WordPress Theme

Современная тема WordPress для djstudio с темным дизайном и hero блоком.

## Автоматический деплой с GitHub

Тема настроена для автоматического деплоя с GitHub на Hostinger:

- **Репозиторий:** https://github.com/Antongotry/djstudio
- **Ветка:** main
- **Путь установки:** /wp-content/themes/djstudio

После каждого push в ветку `main` тема автоматически обновится на сайте.

## Ручная установка

1. Скопируйте папку темы в директорию WordPress:
   ```
   wp-content/themes/djstudio/
   ```

2. Активируйте тему в админ-панели WordPress:
   - Перейдите в `Внешний вид` → `Темы`
   - Найдите "djstudio" и нажмите "Активировать"

## Структура темы

```
djstudio/
├── style.css          # Основные стили темы
├── functions.php      # Функции темы
├── index.php          # Главный шаблон с hero блоком
├── header.php         # Шапка сайта
├── footer.php         # Подвал сайта
├── js/
│   └── main.js        # JavaScript для интерактивности
└── README.md          # Документация
```
